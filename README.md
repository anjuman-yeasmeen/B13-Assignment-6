# 🚀 DigiTools Store

### _A modern digital tools buying platform_

**DigiTools Store** is a premium, high-performance web platform designed for creators, developers, and businesses to explore and purchase digital assets seamlessly. Built with a focus on speed, responsiveness, and a clean user interface.

## 🛠 Tech Stack

This project is built using modern frontend technologies:

- **Frontend Library:** React.js
- **Styling:** Tailwind CSS (Utility-first CSS)
- **UI Components:** DaisyUI
- **Icons:** React Icons & Icons8
- **Deployment:** Netlify / Vercel

---

## ✨ Key Features

- **📱 Optimized Mobile Layout:\*** **📱 Fully Responsive Design:** Optimized for all devices, including mobile, tablet, and desktop.
- **⚡ Premium UI/UX:** A sleek dark-themed interface with modern typography and smooth hover effects, providing a high-end "Store" feel.
- **🛒 Organized Resources:** Categorized sections for Products, Company info, and Resources to ensure users find what they need in seconds.

## 🚀 Getting Started

Follow these steps to run the project locally:

1.  **Clone the repository:**

    ```bash
    git clone [https://github.com/anjuman-yeasmeen/B13-Assignment-6.git](https://github.com/anjuman-yeasmeen/B13-Assignment-6.git)
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```
